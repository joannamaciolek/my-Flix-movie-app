myFlix

myFlix is the web application which provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.
myFlix consists of well designed REST API and architected database built using JavaScript, Node.js, Express, and MongoDB.
The server­side of the myFlix application processes user requests & performs operations against the data stored in the database.

Create new user /users/ POST A JSON object holding data about user to add with following structure:
{
username: 'Jonathan Smith',
password: 'Movie1',
email: 'johnsmith@gmail.com',
birthday: '19.03.1987'
} 		JSON object holding data about user account that was just created. Example:
{
username: 'Martha Nowak',

}
Get/Display user profile /users/[Username] GET A JSON object holding data required to log in.Example:
{
username: 'Jonathan Smith',
password: 'Movie1',
} 		JSON object holding data about user account with the following structure:
{
username: 'Jonathan Smith',
password: 'Movie1',
email: 'johnsmith@gmail.com',
birthday: '19.03.1987'
}
Update user information (username, password, email, date of birth) /users/[Username] PUT A JSON object holding data about user to be updated with following structure:
{
username: 'Martha Nowak',
password: 'Movie2',
email: 'martanowak@gmail.com',
birthday: '20.05.1980'
} 		A text message indicating whether the user info was successfully updated
Add a movie to user's list of favorites /users/favourites_list/[Username]/[Title] POST A JSON object holding data about movie title to be added to user's list of favourites.Example:
{
title:'The Matrix'
}
 JSON object holding data about updated user's list of favourite movies.Example:
{
favourites_list: [ 'About a boy' , 'Amelie' , 'The Matrix']
}
Remove a movie from user's list of favorites /users/favourites_list/[Username]/[Title] DELETE A JSON object holding data about movie title to be removed to user's list of favourites.Example:
{
title: '101 Dalmatians'
}
 A text message indicating that the movie was successfully removed from list of favourites.
Delete user profile /users/[Username]/ DELETE A JSON object holding data about user to be deregistered with following structure:
{
username: 'Jonathan Smith',
password: 'Movie1',
email: 'johnsmith@gmail.com',
birthday: '19.03.1987'
}
 A text message indicating that user was successfully deregistered.
