myFlix

myFlix is the web application which provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.
myFlix consists of well designed REST API and architected database built using JavaScript, Node.js, Express, and MongoDB.
The server­side of the myFlix application processes user requests & performs operations against the data stored in the database.
